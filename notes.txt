trading bot project

goals:
- fetch ohlc data from yahoo finance
- calculate technical indicators (rsi, macd, bollinger, atr)
- build strategies that combine multiple signals
- backtest strategies against historical data

architecture:
- ohlc.py: data fetching
- indicators.py: technical indicator calculations
- strategy modules: individual strategy implementations
- backtest.py: run strategies against historical data

data sources:
- yahoo finance api (free, no key needed)
- possibly add other sources later

timeline:
- december: get data pipeline working, basic indicators
- january: first strategies (gapup, movo, bcross)
- february+: iterate and improve

backtesting notes:
- gapup: signals are rare, need multi-year data to validate
- bcross: stricter than gapup (requires macd crossover not just turning)
- both rely on 52wk low proximity, may cluster in drawdowns
- need to track win rate and avg return per signal
- consider adding holding period analysis (5d, 10d, 20d returns)
